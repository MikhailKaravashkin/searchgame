# Запуск на реальном iPhone

## Шаги для запуска на смартфоне

1. **Открой проект в Xcode**:
```bash
open /Users/miguel/searchgame/SearchGame.xcodeproj
```

2. **Подключи iPhone по USB** к Mac

3. **Настрой подпись в Xcode**:
   - В Project Navigator (слева) кликни на `SearchGame` (синяя иконка проекта сверху)
   - Выбери таргет `SearchGame` в центральной панели
   - Перейди на вкладку **"Signing & Capabilities"**
   - Поставь галочку **"Automatically manage signing"**
   - В поле **"Team"** выбери свой Apple ID (или добавь через "Add Account...")

4. **Измени Bundle Identifier** (если нужно):
   - В том же разделе найди поле **"Bundle Identifier"**
   - Если видишь красную ошибку про конфликт, добавь свой суффикс:
     ```
     com.searchgame.app  →  com.yourname.searchgame
     ```

5. **Выбери устройство**:
   - В верхней панели Xcode рядом с кнопкой ▶️ выбери свой iPhone (не симулятор)

6. **Запусти** (⌘R или кнопка ▶️)

## Первый запуск на устройстве

При первом запуске может появиться ошибка **"Untrusted Developer"**:

1. На iPhone открой: **Настройки → Основные → VPN и управление устройством**
2. Найди свой Apple ID в разделе "Профиль разработчика"
3. Тапни **"Доверять ..."**
4. Вернись в Xcode и запусти снова

## Если остались проблемы

- Убедись что iPhone **разблокирован** при запуске
- Проверь что **доверие к компьютеру** настроено (при первом подключении iPhone спросит)
- Если ошибка про provisioning profile — попробуй изменить Bundle Identifier
